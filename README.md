# Flash-Non-Flash-Fingerphoto-Database
The Flash-Non-Flash Fingerphoto Database (FNF Database) is a two-session contactless fingerprint dataset consisting of paired flash and non-flash images collected using a controlled smartphone-based acquisition setup for non-commercial academic research to aid analysis and evaluation of contactless fingerprint recognition methods.
